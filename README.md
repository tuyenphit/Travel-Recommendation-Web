# Travel-Recommendation-Web
1.Descriptions of projects:

- Name: Travel Recommendaion Web.
- This is a website which recommends users countries, cities, and travel attractions.
  The website must dynamically load travel data, allow users to search or filter destinations, and display personalized recommendations.
- This project demonstrates mastery of:

* JavaScript fundamentals
* DOM manipulation
* Event handling
* Fetching JSON data (async/await)
* Basic UI design with HTML/CSS

2. How to run locally

- Clone the repository.
- Open index.html in a web browser.

3.  Technologies used

- HTML5
- Javascript(ES6, DOM, Fetch API)
- CSS
- JSON for data storage.

4. Data sources

- Images: https://www.pexels.com
- Data : https://chatgpt.com and https://www.wikipedia.org.
